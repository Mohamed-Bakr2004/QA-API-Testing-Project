 API Testing Project - GitHub API

This project showcases my skills in API testing using Postman. It covers CRUD operations (Create, Read, Update, Delete) on GitHub Repositories.

 Technologies Used
- Postman
- GitHub REST API

 What I tested
- **POST:** Creating a new repository.
- **GET:** Retrieving repository details.
- **DELETE:** Deleting the repository.

 How to use
1. Download the `API_Testing_Project.json` file.
2. Open Postman.
3. Click on **Import** in the top left corner.
4. Drag and drop the file.
5. Add your own **Bearer Token** in the Authorization tab to run the requests.
